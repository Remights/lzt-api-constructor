# TOUR_QA — чеклист ручной проверки spotlight-тура

Перед проверкой: **Ctrl+Shift+R**, в консоли `localStorage.removeItem('lzt_tour_done')`, перезагрузка.

## Запуск

- [ ] Тур стартует при первом визите (или через кнопку «Обучение» в настройках, если есть)
- [ ] `window.LZTTour.start()` в консоли открывает тур без ошибок в консоли

## Шаг 1 — язык

- [ ] Модалка **строго по центру** экрана
- [ ] Кнопки RU / EN в одной строке, активная подсвечена зелёным
- [ ] «Шаг 1 из 13» и точки прогресса по центру, без смещения
- [ ] Смена языка меняет текст тура дальше

## Шаг 2 — welcome

- [ ] Модалка по центру (без смещения относительно шага 1)
- [ ] Кнопки «Пропустить» слева, «Далее» справа

## Шаги 3–6 — canvas / блоки

- [ ] Подсветка canvas / кнопки «+» / меню / блок запроса
- [ ] Зелёная обводка пульсирует, без «дёрганья» при движении мыши
- [ ] Шаг «Соедините»: порты мигают, **обводка не перезапускается** при движении курсора без drag

## Шаг 7 — connect (критичный)

- [ ] Drag от out «Старт» к in «Запрос» работает
- [ ] После соединения — praise + «Далее»
- [ ] Нет постоянного мигания/перезапуска анимаций при hover

## Шаги 8–12 — demo / run / log / debug / script

- [ ] Подсветка нужных панелей и кнопок

## Шаг 13 — done
