# LZT API Constructor — для жюри конкурса

No-code клиент **Market + Forum API** Lolzteam: визуальный граф сценариев, прогон в приложении, экспорт бота на 7 языках.

## Чем это не «ChatGPT за минуту»

- Каталог эндпоинтов из **OpenAPI** (Market + Forum)
- Визуальный граф (условия, фильтры, циклы, снайпер, чекер)
- Реальный **прогон** сценария в приложении (с dry-run для покупок)
- **Codegen ×7**: Python / async / Node / Bash / PHP / C# / Go
- Тесты + CI, SSRF-защита локального прокси

## Готовый продукт (первый экран сайдбара)

1. **Поиск дешёвых Steam** — GET → фильтр → лог  
2. **Bump лотов** — `POST /user/bump`  
3. **Чекер лота** — поиск → `item_id` → проверка «не продан»  

Плюс **«Демо за 1 мин»**: поиск → фильтр → dry-run снайпер (mock API) → смотрите Python справа.

Путь для записи: открыть демо → Запустить → скопировать Python из панели «Бот».

## Запуск

```bash
pip install -r requirements.txt
python main.py
```
