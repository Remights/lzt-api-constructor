          Настройка доступа к API

          Для выполнения реальных запросов к Маркету или Форуму необходим персональный Bearer-токен от вашего аккаунта Lolzteam.

          Настройки

          Скрыть ненужные блоки — освободится место для остальных.

          Внешний софт шлёт JSON сюда → сценарий гоняет блоки → ответ обратно. В сценарии данные в vars.hook, ответ положи в vars.hook_response.

          Скрипты (.py/.js):

          Импорт из curl

          Вставьте curl-команду (например, скопированную из DevTools → Network → Copy as cURL) — мы разберём метод, URL, заголовки и тело в готовый блок «Запрос».

          AI-ассистент сценариев

          Опишите задачу простым языком — ассистент соберёт готовый сценарий из блоков. Например: «каждые 5 минут ищи стим дешевле 100р, фильтруй по цене и шли уведомление в телеграм».

          Ключ хранится только локально и уходит напрямую на ваш endpoint.
