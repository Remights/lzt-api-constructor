# Security

## Модель угроз

Приложение — **локальный desktop-клиент**. Сервер слушает только `127.0.0.1`.

| Риск | Митигация |
|------|-----------|
| SSRF через `/api/test` | `backend/security.py` — блок private/localhost URL |
| Утечка API-токена | Только localStorage; опционально AES-GCM (настройки) |
| Утечка AI-ключа | Не хранится; передаётся только в `/api/ai` на выбранный провайдер |
| Произвольные ссылки | `/api/open-browser` — только `http/https` |

## Сообщить об уязвимости
