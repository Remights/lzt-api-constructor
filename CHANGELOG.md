# Changelog

Заметные изменения проекта LZT API Constructor.

---

## [1.1.0] — 2026-07-07

Крупное обновление: демо без токена, анализ цен, блок «ИИ» в сценариях, интерактивное обучение и заметная полировка интерфейса.

### Новое

**Демо и онбординг**
- Режим **«Демо»** — можно попробовать конструктор без API-токена: mock-лоты, mock-запросы и mock-ИИ
- Готовые шаблоны «Демо» и «Умный снайпер» в сайдбаре
- **Spotlight-обучение** — 13 шагов с подсветкой кнопок и панелей; вы сами добавляете блок, соединяете порты, запускаете демо из списка примеров
- На пустом холсте — быстрые чипы: Демо / Умный снайпер / Анализ цены
- Подсказки для новичков: диалог при запуске без связей, напоминание соединить блоки, кнопка «Попробовать демо» в пустых «Мои сценарии»

**Анализ цены лота**
- Минимум, средняя, медиана, вердикт «Выгодно / Дорого»
- Похожие лоты и наглядные полоски цен
- Панель справа и popup при вставке ссылки `lzt.market/…` (в том числе вне холста)
- Бейдж «−X% от медианы»

**Блок «ИИ» в сценарии**
- Оценка до 50 лотов одним промптом, выходы Успех / Ошибка
- Работает в редакторе, при запуске сценария и в генерации кода

**Шаринг и экспорт**
- Обмен сценариями кодом `LZT1:…` и QR-кодом, импорт по коду
- Экспорт лога прогона в `.txt`
- ZIP-экспорт бота привязан к выбранному языку в панели «Скрипт» (Python, Node, Go, C#, Bash, PHP и др.)

**Интерфейс**
- Правая панель: Запуск, Debug, Анализ цены, Скрипт — с настраиваемыми сплиттерами и сохранением размеров
- В настройках можно скрыть Debug / Price / Script
- **Подсказки параметров запроса** — чипы под URL по OpenAPI и общим параметрам Market/Forum, обновляются при смене адреса
- Вставка `lzt.market/ID` на холст → автоматически создаётся блок «Запрос»
- Подсветка активных блоков и рёбер при Run; тост и звук при успешной «покупке» в демо

**AI-ассистент и код**
- Улучшены генераторы **Bash, C#, Go**; в PHP добавлен полноценный блок «Фильтр»
- Semver в автообновлении (`1.0.10` корректно новее `1.0.9`), опция «Не напоминать эту версию»

### Исправлено

- Светлая тема: читаемый текст на вкладках, в toolbar, у кнопки «Справочник», видимые скроллбары
- Белый текст при наведении на кнопки — больше не «пропадает»
- Переводы блока «ИИ» в меню «Добавить блок» (RU/EN)
- Зелёная линия связи не просвечивает через меню «+ Добавить блок»
- Таблица Debug: колонки не наезжают друг на друга
- Блоки не исчезают при pan холста
- Blur модалок, правые сплиттеры, throttle minimap/edges
- Обучение: модалки по центру, без дёрганья подсветки на шаге «Соедините блоки»

### Для разработчиков

- Тур вынесен в модули `web/js/tour/` и `web/css/tour.css`
- Редакторы свойств блоков — в `web/js/scenario/prop_editors/`
- Тесты тура и обновлённый CI
- Чеклист ручной проверки тура: `docs/TOUR_QA.md`

## [1.0.0] — 2026-07-04

Первый публичный релиз.

- Визуальный конструктор сценариев для LOLZTEAM Market и Forum API
- Блоки: запрос, фильтр, условие, снайпер, уведомления, циклы, под-сценарий и др.
- Генерация бота на 7 языков
- Desktop-приложение (PyWebView), автообновление
- AI-ассистент, импорт из curl, встроенный справочник API
- CI и автотесты
